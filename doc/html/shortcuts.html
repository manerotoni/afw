Application shortcuts

Main window


Shortcut Action
F1 Open help browser
F5 Refresh gallery view
CTRL-A Select all items 
CTR-I Invert selection
CTRL-O Open a data file
CTRL-C Close a data file
CTRL-Q Quit the application
ALT-SHIFT-S Toggle Sortingdock
ALT-SHIFT-A Annotationdock
ALT-SHIFT-C Contrastenhancement dock
SHIFT + mouse wheel Zoom

m Maximize image to fit in window
f Fit image to window (expanding)
o Zoom to original size
- Decrease zoom factor
+ Increase zoom factor

Matplotlib navigation shortcuts

For plots CellAnnotator uses matplotlib and it is possible to interact with the plots using matplotlib navigation shortcuts.


Shortcut Action
t show/hide navigation tool bar
h or r or home Home/Reset
c or left arrow or backspace Back
v or right arrow Forward
p Pan/Zoom
o Zoom-to-rect
ctrl + s Save
ctrl + f Toggle fullscreen
ctrl + w Close plot
hold x when panning/zooming with mouse Constrain pan/zoom to y-axis
hold y when panning/zooming with mouse Constrain pan/zoom to x-axis
hold CTRLwhen panning/zooming with mouse Preserve aspect ratio
g when mouse is over an axes Toggle grid
L or k when mouse is over an axes Toggle x axis scale (log/linear)
l when mouse is over an axes Toggle y axis scale (log/linear)
