Application shortcuts

Main window

F1 open help browser
F5 refresh gallery view
CTRL-A select all items 
CTR-I invert selection
CTRL-O open a data file
CTRL-C close a data file
CTRL-Q quit the application
ALT-SHIFT-S toggle Sorting dock
ALT-SHIFT-A toggle Annotation dock
ALT-SHIFT-C toggle Contrast enhancement dock
SHIFT + mouse wheel zoom

Data Importer Dialog/Segmentation settings dialog

F5 refresh view (update segmentation, bounding boxes, ...)


Matplotlib
For plots  CellAnnotator uses matplotlib. Therefore it is  possible to  interact with the plots using the same  shortcuts

t enables disables the toolbar at the bottom of the plot
