Getting started

Graphical User Interface (GUI):

The graphical user interface contains a menu bar, tool bars for sorting/ranking, display options and data loading. There is a separate tool bar at the bottom for navigation within a cellh5 file.  Dock widgets (sidebars)  for Annotation, Sorting and Contrast enhancement supply most of the functionality. All tool bars and docks can be toggled using the View menu. The gallery- or thumbnail viewer is the central part of CellAnnotator. Its purpose is to display all foreground or cell objects. The gallery viewer is not displaying a stitched image, it is displaying a real thumbnail gallery where each image is a click able and selectable item.  After loading the items are unordered as they appear in the data file.
The following sections describe how to open data files, select items and how to sort the items using the context menu.

Open a data file

There are  three ways to open a data file:
  1. Menu: File->Open
  2. Press the open-icon  in the tool bar
  3. Drag & drop: drag the data file from the file manager  and drop it in the main window (anywhere, except in the central gallery viewer)
A progress bar and a "Abort" button will appear at the bottom of the main window.  Loading of data causes heavy IO and may block the GUI.  CellAnnotators own hdf data model is optimized for fast loading. Size of the thumbnails is predefined and all thumbnails in the files will be loaded. In contrast, Cellh5 does not support thumbnails, they are generated during the loading process, which is time consuming. Size of the gallery images is alterable and the number of items that are loaded can be changed (default is 500), since Cellh5 is a storage format for HCS-data.

Select items - working with the gallery viewer:

To select an item, simply double click on the item. To select multiple items press the "CTRL"-button and click on the items of  interest consecutively.  It is also possible to drag the mouse over  several items while the "CTRL"-button is pressed.


The gallery view has a context Menu. Note the sub menu "throw sort anchor", which is a efficient an fast way for sorting the gallery view.

refresh Updates the display (shortcut "F5")
select all Select all items in the gallery view
invert selection Invert the selection in the gallery view
throw sort anchor Sort the items by cosine similarly, the currently selected item(s)
serve as reference example for the similarity measure

Sorting items (quick sort):

Sorting occurs linearly from the upper left to the bottom right corner. It is necessary to define at least one reference example for sorting. After sorting the reference example will be positioned always in the upper left corner of the view. CellAnnotator uses by default the cosine similarity a measure. If multiple items are selected,  the arithmetic mean is used  as reference example.

To quick sort  the items in the gallery view do:
  1. Select at least one item in the gallery view.
  2. Right click in the gallery view to open the context menu.
  3. Click on the menu "throw sort anchor".
