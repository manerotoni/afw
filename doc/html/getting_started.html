Getting started

Graphical User Interface (GUI):

The graphical user interface contains a menu bar, toolbars for sorting/ranking, display options and loading data. There is a separate toolbar at the bottom for navigation within a cellh5 file.  Dockwidgets (sidebars)  for Annotation, Sorting and Contrast enhancement supply most of the functionallity. All toolbars and docks can be toggled using the View menu. The gallery- or thumbnail viewer is the central part of CellAnnotator. Its purpose is to display all forground or cell objects. The gallery viewer is not displaying a stiched image, it is displaying a real thumbnail gallery where each image is a clickable and selectable item.  After loading the items are arranged arbitraily in the order the appear in the data file.
The following sections describe how topen data files, select forground objects or items and how to sort those items using the context menu.

Open a hdf or cellh5 file: 

There are  three ways to open a data file:
  1. Menu: File->Open
  2. Press the open-icon  in the toolbar
  3. Drag & drop: drag the hdf-/ cellh5 file from the file manager  and drop the file in the main window (anywhere, except in the central gallery viewer)
A progress bar  in the status bar of the main window will appear  beside an abort button.  Loading of data causen heavy IO of the hard disk and may block the GUI unit loading is finished.  CellAnnotators own hdf data model is optimized for fast loading. Size of the thumbnails is predefined and all thumbnails in the files will be loaded. In contrast, Cellh5 does not support thumbnails, they are generated during the loading process, which is time consuming. Size of the gallery images is alterable and the number auf items that are loaded can be changed (default is 500), since Cellh5 is a storage formate for HTS-data.


Select items - working with the gallery viewer:

To select an item simple double click on an item. To selcect multiple items press the "CTRL"-button and click on the items of  interenst consecutively.  It is also possible to drag the mouse over  several items while the "CTRL"-button is pressed.



The gallery view has a context Menu. Note the sub menu "throw sort anchor" which is a efficient an fast way for sorting the gallery view. See the table below for a detailed explaination.

refresh
 updates the display (shortcut "F5")
select all select all items in the gallery view
invert selection invert the selection in the gallery view
throw sort anchor Sort the items by consine similary, the currently selected item(s)
serve as reference example for the similarity measure


Sorting items (quick sort):

Sorting occurs linearly from the upper left to the bottom right corner. It is necessary to define at least one reference example for sorting. After sorting the reference example will be positioned always in the upper left corner of the view. CellAnnotator uses by default the cosine similarity a measure. If multiple items are selected,  the arithmetic mean is used  as reference example.

To quicksort  the items in the gallery view do:
  1. select at least one item in the gallery view
  2. right click in the gallery view to open the context menu
  3. click on the menu "throw sort anchor"
